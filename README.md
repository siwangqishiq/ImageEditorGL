## ImageEditorGL
基于OpenGLES-3.0 实现的图片编辑库, 利用了移动设备的GPU能力 完成图片的编辑 渲染操作

### 项目技术栈
    kotlin 原生Android UI 展示性控件
    c++ MainView 主编辑视图
    glsl es3.0 shader 图片编辑 渲染的实现
